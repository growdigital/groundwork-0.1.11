Styleguide

A guiding principle in Kanban) process management is make work visible.

This is why a styleguide is so important.

Groundwork comes bundled with Fractal, a magnificent, automated, styleguide builder. So long as your module folder has an HTML template file (we use Handlebars), then the module will appear in your styleguide.

Big kudos to Pattern Lab by Brad Frost – very inspirational work. I find the idea attractive but I don’t get on with the layered granularity. Worth a look though :)
