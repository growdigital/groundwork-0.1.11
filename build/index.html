Quick and easy Modular CSS

A quick way to start writing modular CSS, Groundwork is easy to install and comes complete with a fully-fledged static site generator and styleguide.

Make sure to have Node.js and git installed. Run these commands in your terminal:

$ git clone git@github.com:growdigital/groundwork.git
$ cd groundwork
$ npm install
$ npm run dev

See the posts for a step-through tutorial on using Groundwork, or visit the website for some background information: www.groundwork.rocks

Make it your own

Change the package.json and index.js to make the project truly yours. And start styling it away from these intentionally drab and dreary colours:

white
gainsboro
gray
dimgray
black

Groundwork is for web designers and frontend developers who are comfortable with the terminal and who want to wrap their heads around the how to Modular CSS works.
