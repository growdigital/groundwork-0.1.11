semver setup

Sat 25 Mar 2017

I just set up semantic versioning for Groundwork with semver.org, using the interactive guide to package.json as a guide. I like semantic versioning, it appeals to my tidy side. Also, it makes a great deal of sense for any sane method of development, web or whatever.

Along the same lines, I can also heartily recommend The Twelve-Factor App, casts a big shining light on all this plugin semantic versioning for Metalsmith.
