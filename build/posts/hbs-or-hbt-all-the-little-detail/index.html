hbs or hbt – all the little detail

Fri 24 Mar 2017

I wasn’t sure if the file extension for Handlebars is .hbt or .hbs, so I asked on the friendly Metalsmith Slack channel, and an answer came back from Woody:

I think it's .hbs because by definition handlebars is a template so .hbt is a but of a misnomer.

It’s all these little details that take the time. And I haven’t even started writing the documentation.
