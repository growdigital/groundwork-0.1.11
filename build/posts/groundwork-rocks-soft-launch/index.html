groundwork.rocks soft launch!

Thu 23 Mar 2017

Groundwork is a Module CSS Starter Kit. Make sure to have Node.js installed, download the repo, then just run this in a terminal:

$ npm run dev

It’s so new that it doesn’t even have semantic versioning yet.

The website www.groundwork.rocks unofficially went live yesterday. I registered the domain with the excellent iwantmyname.com folks, and it’s deployed on Netlify. There’s even a badge for it:
